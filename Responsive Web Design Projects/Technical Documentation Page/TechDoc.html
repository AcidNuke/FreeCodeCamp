Introduction

Project Building is, and has always been a struggle for most. Many people find themselves in a space where they are unsure where and what to start with. How to setup the base and how to finish it off. This Document serves as a guide to assist those in similar positions as to myself. This document will outline some starting points and how to progress from them.

I will explain some basic ways to get started and look at things that you will need to best assist with the process. Dont feel obliged to follow this guide word for word, use it purely as a reference. I have broken the document into various parts so that it's easier to reference back to a specific point if you get stuck.

Never assume that all documentation is correct, it's only as good as the way you interpret it. This document serves only as a guide and cannot be taken as the ultimate, as there are various takes and ways to actually accomplish the takes at hand. Use this at your own risk.

Always code as if the guy who ends up maintaining your code will be a violent psychopath who knows where you live.
